Readme | rtTimelineApp
---------------------------------------------------------------------------------------------

rtTimelineApp is an simple web application developed using PHP/Twitter API/JQuery


Features
-----------------
1) Authenticating an user using oAuth

2) Showing 10 latest tweets from the user timeline in a jquery slider

3) An option to download the tweets as a PDF

4) Show any 10 random followers of the authenticated user.

5) Search follower box ( with auto- suggest)

6) On clicking the follower name, the jquery slider will show 10 tweets of the selected follower
